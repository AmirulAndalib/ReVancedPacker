# ⚒️ Building
TIP: You can add UPX to reduce the app size!
```
git clone https://github.com/xemulat/ReVancedPacker
cd ReVancedPacker
pip install -r requirements.txt
pip install pyinstaller
pyinstaller --clean --onefile main.py
```
